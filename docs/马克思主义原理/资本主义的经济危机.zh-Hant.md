# 資本主義經濟危機
資本主義經濟危機的根本原因是資本主義基本矛盾的尖銳化，這具體表現為生產過剩和消費不足的矛盾。當資本主義發展到某個階段，經濟危機便不可避免，其主要特徵是生產過剩。

1. **生產過剩**：資本主義經濟危機的本質特徵是生產過剩。這種過剩是相對過剩，即相對於勞動人民有支付能力的需求而言，生產的商品顯得過剩，而不是與勞動人民的實際需要相比的絕對過剩。
2. **貨幣流通與信用危機**：經濟危機的可能性由貨幣作為流通手段和支付手段所引起。商品買賣時間上的分離，以及賒購賒銷方式下的信用破壞，都是危機在形式上的可能性。
3. **根本原因**：資本主義經濟危機爆發的根本原因在於資本主義的基本矛盾，具體表現為生產無限擴大的趨勢與勞動人民有支付能力的需求相對縮小的矛盾，以及單一企業內部生產的有組織性和整個社會生產的無政府狀態之間的矛盾。
